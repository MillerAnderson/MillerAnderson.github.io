Hexo博客搭建全记录

2017-03-02
  • Hexo

在本地安装Hexo
1.安装Node.js
2.在E盘新建一个文件夹Hexo
3.按如下步骤通过npm安装Hexo


10 更改主题:
1>下载hueman主题
2>解压放到blog/themes目录下面
3>删除.github文件夹及.gitignore文件
4>修改blog目录下的_config.yml: theme: hueman 注意冒号后面有个空格
5>hexo g
6>hexo s
11.发布到github
0>在github新建一个分支milleranderson.github.io(注意:milleranderson是账户名)
1>修改blog目录下的_config.yml
deploy:
type: git
repo: https://github.com/MillerAnderson/MillerAnderson.github.io.git
branch: master
2>设置git(在Git Bash命令行模式下)
01.$ git config –global user.name “milleranderson”
02.$ git config –global user.email “67151970@qq.com”
03.$ ssh-keygen -t rsa -C “67151970@qq.com” (我自己连续三个回车,没有设置密码)
04.$ eval “$(ssh-agent -s)” 查看得到结果Agent pid 2636
05.$ ssh-add ~/.ssh/id_rsa 添加ssh key到 ssh-agent
06.登录github添加ssh key: 将刚才生成的密钥对中的id_rsa.pub公钥中的全部内容添加进去
07.$ ssh -T git@github.com 测试
3>安装 $ npm install hexo-deployer-git –save
4>在blog/public目录下添加一个文件”CNAME”,里面的内容是”andyself.com”
5>在域名的DNS中添加三条记录(前两条是指向github)
@ A 192.30.252.153
@ A 192.30.252.154
www CNAME milleranderson.github.io.
5>hexo d
12.网站添加评论与计数
1>登录”多说”,在后台管理,将andyself.duoshuo.com中的andyself
2>设备theme下面的_config.yml
duoshuo: # enter duoshuo shortname here =>duoshuo: andyself
并将默认的disqus注释或者删除
3>添加计数(使用不蒜子:http://ibruce.info/2015/04/04/busuanzi/)
01.在theme/layout/common目录下的head.ejs的标签里添加一行

02.可在需要的地方添加本站pv/uv及本面pv

本站总访问量次


本站访客数人次


本文总阅读量次

注: 在common目录下还可以看到其它ejs文件,其实这些是用于生成html文件的模板,
将上面的点击量显示放在你希望显示的模板的某一区域即可
另: 有些模板不是在common目录下,自己搜索一下文件名比如head.ejs即可找到该文件夹
13.为文章添加图片
1>在source文件夹下新建image文件夹,并放入图片
2>在文件中通过 来调用
14.创建”About”页面: hexo new page about
15.创建文章: hexo new xxxxx
